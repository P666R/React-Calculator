# Installation And Usage:

## To run this project run

-  git clone https://github.com/P666R/React-Calculator
  
## Go to directory

-  cd React-Calculator-main
  
## Install all dependencies

- npm install
 
## Run Project

-  npm start
  
## Now you can access on localhost:3000
